# Blueprint SkillForge

A hands-on fullstack project built with **React + TypeScript**, **Kotlin & Spring Boot**, **PostgreSQL**, **AWS**, and **Docker**.  
The goal: explore modern technologies together, practice cloud & DevOps, dive into authentication (OAuth2 + JWT), and experiment with AI features ‚Äì all while building something useful and fun.
